Welcome to HTWG MONOPOLY

This is a webversion of the existing HTWG-Monopoly found here: HTWG-Monopoly on GitHub

Why

This project is part of an assignment of the lecture Webtechnologies at the HTWG in Konstanz.

How

This project is build with various technologies in order to cover up some of the techniques how to provide a web application. Used technology:

HTML & CSS Bootstrap Play framework (scala) Angular jQuery & Ajax Javascript WebSockets Polymer Heroku

This project is still under construction and will eventually go online and will be playable with a browser.
